API

How to use
https://nsk-1010.github.com/5000choyen/api.html?top=上面的字符串&bottom=下面的字符串&correctjp=yes或no
name value description
top - 上面的字符串
bottom - 下面的字符串
bx - 底线左侧的空白空间（单位：px）
hoshii true/false 底下的字符串将被固定为"欲しい!"
alpha true/false 使背景颜色透明
correctjp yes/no 使用可疑的日文
